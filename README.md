## How to run app locally

```
git clone https://github.com/SK02K1/Video-Library.git
cd Video-Library
npm install
npm start
```

## Features

- Homepage
- Video Listing Page
- History Management
- Playlist Management
- Likes Management
- Watch Later Management
- Authentication
  - Login
  - Logout
  - Sign-up
- Search Video
- Single Video Page
- Filter Videos with Category
- Loading & Alerts
- User Profile Page
